## Cache

### 요구사항
- GreetingControllerTest 클래스의 모든 테스트 메서드를 통과시킨다.

### 학습순서
- 테스트 코드로 HTTP 헤더를 검증하는 방법을 익힌다.
- HTTP 압축을 설정한다.
- HTTP 헤더에 Cache-Control를 적용하고 캐싱이 적용됐는지 확인한다.
- 캐시 무효화(Cache Busting)를 학습한다.
